# CORTA-Maintenance

AI-powered predictive maintenance platform for industrial production lines.

## Features

- 10 AI-powered modules for proactive maintenance
- 18+ core platform modules covering full CMMS functionality
- Real-time sensor analytics and failure prediction
- Smart work order triage and technician assignment
- Comprehensive safety, compliance, and audit management

## Development

```bash
bun install
bun run dev
```

## Docker

```bash
docker compose up -d
```

Access at `http://localhost:7701`
